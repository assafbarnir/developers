To populate your Qordoba account with your project content, paste the following code into the page you want to track inside the <head> and </head> tags, before the first section. Change site-key to—you guessed it—your site key. You can find your key in the Settings tab in the Qordoba web application.

<script type="text/javascript">
  document.write("<scrip"+"t src=\"//qcdn.qordoba.com/qordoba-	latest.min.js\"></scrip"+"t><scrip"+"t src=\"https://storaged.qordoba.com/sdk-settings-<site-key>.js?cachebust="+Math.random()+"\"></scrip"+"t>")
</script>

The SDK works with all major frontend frameworks, including Angular and Backbone.
