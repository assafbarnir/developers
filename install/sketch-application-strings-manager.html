Installation

  1. Install Qordoba

    After downloading the package, execute:

     $ sudo yum install epel-release
 $ sudo yum install qordoba-community-edition-0.2.1.aws.rpm --nogpgcheck

  2. Prepare your database

    Configure Qordoba so it can connect to your database. Qordoba supports both PostgreSQL 9.5+ and Cassandra 3.x.x as its datastore.

    If you are using Postgres, please provision a database and a user before starting Qordoba, i.e.:

     CREATE USER qordoba; CREATE DATABASE qordoba OWNER qordoba;

    Now, run the Qordoba migrations:

     $ qordoba migrations up [-c /path/to/qordoba.conf]

    Note: migrations should never be run concurrently; only one Qordoba nodes should be performing migrations at a time.

  3. Start Qordoba

     $ qordoba start [-c /path/to/qordoba.conf]

  4. Use Qordoba

    Qordoba is running:

     $ curl -i http://localhost:8001/
